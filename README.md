# WHU-thesis-template-word

这是一份武汉大学本科毕业论文（设计）模板，采用2024年的新规范制作

## 自定义样式

- 正文
- 封面
- 目录
- 一级标题
- 二级标题
- 三级标题
- 图注
- 表注
- 图片
- 表格（默认三线表）
- 页眉
- 页脚

实现了一些特定样式的自动编号

有新的问题欢迎反馈，一起打造更好用的毕设模板
